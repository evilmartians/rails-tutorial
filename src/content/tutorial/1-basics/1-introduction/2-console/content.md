# Run Ruby REPL

You can also play with Ruby in your terminal! For that, run the following command:

```sh
bin/console
```

Try executing the following _spells_ in the console to see some weird Ruby stuff:

```rb
1::+(2)

str = ""
str << 97
# what is str?
#
?a + ?1
```
